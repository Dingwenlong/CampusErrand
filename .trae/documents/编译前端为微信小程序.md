# 编译前端为微信小程序计划

## 项目分析
- **目标项目**：campus-errand-uniapp（基于UniApp框架开发的校园跑腿小程序）
- **已配置项**：
  - 微信小程序编译脚本：`build:mp-weixin`
  - 微信小程序依赖包：`@dcloudio/uni-mp-weixin`
  - 小程序配置文件：`src/manifest.json`（appid为空）

## 编译步骤

### 1. 准备工作
- 进入 `campus-errand-uniapp` 目录
- 检查并安装依赖（如果需要）

### 2. 配置微信小程序信息
- **可选**：在 `src/manifest.json` 中填写微信小程序的 appid
  - 当前 appid 为空，编译后可在微信开发者工具中配置

### 3. 执行编译命令
- 运行 `npm run build:mp-weixin` 命令
- 编译完成后，生成的代码会位于 `dist/build/mp-weixin` 目录

### 4. 预览和调试
- 打开微信开发者工具
- 导入编译后的 `dist/build/mp-weixin` 目录
- 配置小程序 appid（如果未在 manifest.json 中配置）
- 预览和调试小程序

## 注意事项
- **网络请求**：确保后端API地址可访问，可能需要在微信开发者工具中配置域名白名单
- **权限配置**：manifest.json 中已配置位置权限，编译后会自动应用
- **兼容性**：UniApp 编译的小程序可能需要在微信开发者工具中进行少量调整

## 预期结果
- 成功编译生成微信小程序代码包
- 可在微信开发者工具中正常预览和调试
- 小程序功能与原前端代码保持一致
